    # 283 移动零

    给定一个数组 nums，编写一个函数将所有 0 移动到数组的末尾，同时保持非零元素的相对顺序。

    示例:

    输入: [0,1,0,3,12]
输出: [1,3,12,0,0]
    1
    2

    说明:

    1. 必须在原数组上操作，不能拷贝额外的数组。
    2. 尽量减少操作次数。
    点击查看答案 
    var moveZeroes = function(nums) {
  let n = nums.length;
  let left = 0;
  let right = 0;
  while (right < n) {
    if (nums[right] != 0) {
      let mid = nums[left];
      nums[left] = nums[right];
      nums[right] = mid;
      left += 1;
    }
    right += 1;
  }
};

    直接补零

    var moveZeroes = function(nums) {
  let indexNow = 0;
  let indexNum = 0;
  let m = nums.length;

  while (indexNum < m) {
    if (nums[indexNum] != 0) {
      nums[indexNow++] = nums[indexNum];
    }
    ++indexNum;
  }

  for (let i = indexNow; i < m; i++) {
    nums[i] = 0;
  }
};
